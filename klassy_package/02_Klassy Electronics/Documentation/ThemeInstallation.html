Theme Installation

  • 1) First Go admin panel of magento go to system >> cache mangement and Disable all cache.
  • 2) Upload Theme Files to magento Directory (app folder) using cpanel or ftp accounts.
  • 3) Active Theme Extensions, open command line in folder root of magento and run commands via ssh using putty or others, php bin/magento setup:upgrade.

  • 4) Load Theme Static Content, open command line in folder root of magento and run commands via ssh using putty or others, php bin/magento setup:static-content:deploy.

  • 5) select "Template Trend PowerLab Store " theme in Content > Design > Configuration From Magento.
