1) Newsletter Popup Setting
Go to Magento Admin panel Ortho Theme >> Theme Settings Click On Newsletter Popup.
Below Option are set.
Newsletter Popup Enable: If this value is yes than Newsltter Popup Enabled.
Cookie Name: Name of the Cookie. If you change this than popup agin display to all customer.
Cookie Expires Days: After days show Popup again.


2) Advancemenu Setting
Go to Magento Admin panel Ortho Theme >> Theme Settings Click On Advancemenu Settings.
Below Option are set.
Advance Menu Enable: If this value is yes than Advance Menu Enabled else Default menu enabled.
Home Link Enable: If this value is yes than Home Link Enabled else Home Link diabled.
Static Block Before Position: In order to display a custom block with images or other html content in the before advance menu popup, use the Static Block with the ID like advancemenu_top_block_3 where 3 is the category id..
Static Block After Position: In order to display a custom block with images or other html content in the after advance menu popup, use the Static Block with the ID like advancemenu_bottom_block_3 where 3 is the category id.
Custom Menu Enable: If this value is yes than Custommenu Enabled.
Custom Menu Name: Name Of Custom Menu Which is Displayed on Navigation.
Custom Menu Link: Redirection of custom menu to which page complete url like http://www.domain.com/custom.
Custom Menu Block Id: Display The Static Block On Popup Of Custommenu.


3) Sidebar Column Setting
Go to Magento Admin panel Ortho Theme >> Theme Settings Click On Sidebar Column Setting.
Below Option are set.
Sidebar Category Enable: If this value is yes than Category is Displayed on Sidebar.
Sidebar Common Staticblock Enable: In order to display a custom block with images or other html content in the sidebar, use the Static Block with the ID like left_banner_common.
Sidebar Categorywise Staticblock Enable: In order to display a custom block with images or other html content in the sidebar, use the Static Block with the ID like left_banner_category_3, where 3 is category id.


4) Product Page Settings
Go to Magento Admin panel Ortho Theme >> Theme Settings Click On Product Page Settings.
Below Option are set.
Product page Static block Enable: In order to display a custom block with images or other html content in the product page, use the Static Block with the ID like product_custom_block.
Product page Custom Tab Enable: In order to display a custom block with images or other html content in the tab, use the Static Block with the ID like product_custom_tab.


5) New Product Settings
Go to Magento Admin panel Ortho Theme >> New Product Settings Click On New Product Settings.
Below Option are set.
Module Enable: If this value is yes than New Product Module is Displayed.
Title: Display Title Of New Product Module.
Max. Product On Home Page: Set Maximum Product to displayed On Home Page.
Sidebar Column: If this value is yes than New Product Module is Displayed on the Sidebar.
Max. Product On Side Bar: Set Maximum Product to displayed On Sidebar.
